# How do you optimize bundle size in modern frontend projects?

Optimizing bundle size is crucial for performance. Interviewers want to know if you can identify and fix bundle bloat.

## Why Interviewers Ask This Question

- To check your knowledge of performance optimization
- To see if you can use modern tools and techniques

## Best Practices

- Code splitting and lazy loading
- Tree shaking unused code
- Minification and compression
- Analyze bundles with tools (Webpack Bundle Analyzer)
- Use modern libraries and avoid large dependencies

## Example

```js
// Webpack Bundle Analyzer
const BundleAnalyzerPlugin = require('webpack-bundle-analyzer').BundleAnalyzerPlugin
```

## Interview Tips

- Mention tools and metrics
- Give real-world examples

## Useful English Phrases & Sentence Starters

- To optimize bundle size, I ...
- I use tools like ...
- For example, ...

## Sample Answer

> To optimize bundle size, I use code splitting, tree shaking, and analyze bundles with tools like Webpack Bundle Analyzer. I also avoid large dependencies and use modern libraries for better performance.
