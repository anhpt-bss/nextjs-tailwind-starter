# Share your experience deploying apps to Vercel/Netlify/AWS/S3.

Deployment is a key part of frontend development. Interviewers want to know if you have hands-on experience with modern deployment platforms.

## Why Interviewers Ask This Question

- To check your deployment experience
- To see if you can automate and troubleshoot deployments

## Platforms

- Vercel: great for Next.js, automatic builds, previews
- Netlify: static sites, serverless functions
- AWS/S3: static hosting, scalable, more setup

## Example

- Connect repo, configure build settings, set environment variables
- Use CLI or dashboard for deployment

## Interview Tips

- Mention pros/cons of each platform
- Discuss automation and rollbacks

## Useful English Phrases & Sentence Starters

- I have deployed apps to ...
- For example, ...
- I prefer ... for ...

## Sample Answer

> I have deployed apps to Vercel for Next.js projects, Netlify for static sites, and AWS S3 for scalable hosting. I automate deployments with CI/CD and monitor for issues post-deployment.
