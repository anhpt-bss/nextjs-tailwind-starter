# How do you ensure accessibility (a11y) in your web apps?

Accessibility is essential for inclusive web apps. Interviewers want to know if you can build apps usable by everyone.

## Why Interviewers Ask This Question

- To check your knowledge of a11y best practices
- To see if you can use tools and techniques for accessibility

## Best Practices

- Use semantic HTML and ARIA attributes
- Ensure keyboard navigation
- Test with screen readers
- Color contrast and font size
- Automated a11y testing tools (axe, Lighthouse)

## Interview Tips

- Mention legal requirements (WCAG)
- Discuss testing and tools

## Useful English Phrases & Sentence Starters

- To ensure accessibility, I ...
- For example, ...
- I use tools like ...

## Sample Answer

> To ensure accessibility, I use semantic HTML, ARIA attributes, and test with screen readers. I also use tools like axe and Lighthouse to catch a11y issues early in development.
