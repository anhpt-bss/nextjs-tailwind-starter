# What is Atomic Design? How do you apply it?

Atomic Design is a methodology for creating design systems. Interviewers want to know if you can build scalable, reusable UI components.

## Why Interviewers Ask This Question

- To check your knowledge of design systems
- To see if you can structure UI libraries

## Atomic Design Principles

- Atoms: basic building blocks (buttons, inputs)
- Molecules: groups of atoms (input + label)
- Organisms: complex components (header, form)
- Templates: page structure
- Pages: specific content

## Example Structure

```
src/
  components/
    atoms/
    molecules/
    organisms/
    templates/
    pages/
```

## Interview Tips

- Mention reusability and scalability
- Give real-world examples

## Useful English Phrases & Sentence Starters

- Atomic Design is ...
- I apply Atomic Design by ...
- For example, ...

## Sample Answer

> Atomic Design is a methodology for building UI systems by breaking them into atoms, molecules, organisms, templates, and pages. I apply it to create reusable, scalable component libraries.
