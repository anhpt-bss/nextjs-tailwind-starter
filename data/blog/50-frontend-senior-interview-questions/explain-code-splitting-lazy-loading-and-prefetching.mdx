# Explain code splitting, lazy loading, and prefetching.

These techniques are essential for optimizing frontend performance. Interviewers want to know if you can use them to improve user experience.

## Why Interviewers Ask This Question

- To check your knowledge of performance optimization
- To see if you can use modern build tools

## Code Splitting

- Breaks code into smaller chunks loaded on demand

## Lazy Loading

- Loads components or modules only when needed

## Prefetching

- Loads resources in advance for faster navigation

## Example

```js
// React lazy loading
const LazyComponent = React.lazy(() => import('./LazyComponent'))
```

## Interview Tips

- Mention tools (Webpack, React.lazy, NextJS dynamic)
- Discuss user experience improvements

## Useful English Phrases & Sentence Starters

- Code splitting is ...
- Lazy loading helps ...
- Prefetching improves ...

## Sample Answer

> Code splitting breaks the app into smaller chunks, lazy loading loads them only when needed, and prefetching loads resources in advance. I use these techniques to improve performance and user experience.
