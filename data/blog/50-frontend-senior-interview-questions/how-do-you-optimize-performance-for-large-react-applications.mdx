# How do you optimize performance for large React applications?

Performance optimization is crucial for large React apps. Interviewers want to know if you can identify and fix bottlenecks.

## Why Interviewers Ask This Question

- To check your knowledge of performance best practices
- To see if you can apply optimizations in real projects

## Best Practices

- Code splitting and lazy loading
- Memoization (React.memo, useMemo, useCallback)
- Avoid unnecessary renders
- Virtualization for large lists
- Optimize images and assets
- Use production builds

## Example

```js
const MemoizedComponent = React.memo(MyComponent)
```

## Interview Tips

- Mention tools: React DevTools, Lighthouse
- Be ready to discuss trade-offs
- Give real-world examples

## Useful English Phrases & Sentence Starters

- To optimize performance, I ...
- I use tools like ...
- For example, ...

## Sample Answer

> To optimize performance in large React apps, I use code splitting, memoization, and virtualization for large lists. I also analyze performance with React DevTools and Lighthouse, and optimize images and assets for faster load times.
