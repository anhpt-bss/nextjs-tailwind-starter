# How do you handle SEO in a SPA with NextJS?

SEO is a challenge in single-page applications. NextJS provides features to improve SEO, and interviewers want to know if you can use them effectively.

## Why Interviewers Ask This Question

- To check your knowledge of SEO best practices
- To see if you can use NextJS features for SEO

## Best Practices

- Use SSR or SSG for crawlable content
- Set meta tags with next/head
- Generate sitemaps and robots.txt
- Optimize page speed and accessibility

## Example

```js
import Head from 'next/head'
;<Head>
  <title>My Page</title>
  <meta name="description" content="..." />
</Head>
```

## Interview Tips

- Mention SSR/SSG for SEO
- Discuss dynamic vs static content
- Be ready to give examples

## Useful English Phrases & Sentence Starters

- To improve SEO, I ...
- NextJS helps with SEO by ...
- For example, ...

## Sample Answer

> To handle SEO in a SPA with NextJS, I use SSR or SSG to ensure content is crawlable, set meta tags with next/head, and optimize page speed. NextJS makes it easier to build SEO-friendly single-page applications.
