# What is RxJS? How is it used in Angular?

RxJS is a powerful library for reactive programming with Observables. Interviewers want to know if you can use RxJS effectively in Angular apps.

## Why Interviewers Ask This Question

- To check your knowledge of reactive programming
- To see if you can use Observables in Angular

## What is RxJS?

- Library for reactive programming using Observables
- Allows async data streams, operators, and composition

## RxJS in Angular

- Used for HTTP requests, forms, event handling
- Angular services and components often use Observables

## Example

```ts
import { Observable } from 'rxjs';
this.http.get('/api/data').subscribe(data => { ... });
```

## Interview Tips

- Mention common operators (map, filter, switchMap)
- Discuss why Observables are useful
- Be ready to give code examples

## Useful English Phrases & Sentence Starters

- RxJS is ...
- In Angular, Observables are used for ...
- For example, ...

## Sample Answer

> RxJS is a library for reactive programming with Observables. In Angular, I use RxJS for handling HTTP requests, forms, and events. Observables make it easy to work with async data streams and compose complex logic.
